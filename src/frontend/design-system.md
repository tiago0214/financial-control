# Design System

The design system is based on a "Fresh mint on dark navy" theme.

## Typography

The project uses two primary font families from Google Fonts:
- **Sans Serif (Body)**: `'Inter', sans-serif` (`--font-sans`)
- **Display (Headings)**: `'Space Grotesk', 'Inter', sans-serif` (`--font-display`)

## Gradients

- **Primary**: `linear-gradient(135deg, hsl(156 84% 55%), hsl(168 85% 50%))`
- **Hero**: `linear-gradient(135deg, hsl(156 84% 55%) 0%, hsl(180 75% 50%) 50%, hsl(200 80% 60%) 100%)`
- **Accent**: `linear-gradient(135deg, hsl(14 95% 65%), hsl(35 95% 60%))`
- **Card**: `linear-gradient(180deg, hsl(222 40% 11%) 0%, hsl(222 40% 8%) 100%)`
- **Glow**: `radial-gradient(circle at 50% 0%, hsl(156 84% 55% / 0.25), transparent 70%)`

## Border Radious

- **Large (`--radius-lg`)**: `1rem`
- **Medium (`--radius-md`)**: `calc(1rem - 2px)`
- **Small (`--radius-sm`)**: `calc(1rem - 4px)`
